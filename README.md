# React Counter App – Increment & Decrement using useState

A simple and beginner-friendly React application that demonstrates **state management** using the `useState` hook. The app displays a numeric counter with **Increment** and **Decrement** buttons to update the value dynamically. Built with **Vite.js** and styled using clean, modern CSS.

---

## Overview

This project showcases how React state works by updating a counter value in real time when the user interacts with buttons. It is ideal for beginners who want to understand:

- React state management
- `useState` hook
- Event handling in React
- Basic component-based UI structure

---

## Key Features

- **useState Hook:** Manages and updates the counter value
- **Increment & Decrement Buttons:** Increase or decrease the count instantly
- **Real-time UI Update:** Changes reflect immediately on button click
- **Clean UI:** Simple layout with modern CSS styling
- **Fast Setup:** Built using Vite for quick development

## Installation & Setup

Follow the steps below to run the project locally.

### **Prerequisites**
Make sure you have **Node.js** and **npm** (or yarn/pnpm) installed.

### **1. Clone the repository**
```bash
git clone https://github.com/amna-mohsin/react-counter-app.git
cd react-counter-app
````

### **2. Install dependencies**

```bash
npm install
# or
yarn install
```

### **3. Start the development server**

```bash
npm run dev
# or
yarn dev
```

Open the app in your browser at:
**[http://localhost:5173](http://localhost:5173)**
